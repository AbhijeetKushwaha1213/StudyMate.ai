## Animation Sequences

### Card Hover Animation
```
Normal State:
┌──────────────┐
│     Icon     │
│    Title     │
│  Description │
└──────────────┘

Hover State (0.3s):
┌──────────────┐  ↗️ Scale 1.05
│   Icon ✨    │  ← Glow effect
│  **Title**   │  ← Shadow increases
│  Description │
└──────────────┘
```

### Generation Stages (Each 1.5s)
```
Stage 1: 📖 ← Bouncing
[●○○○] Reading Notes...

Stage 2: 🧠 ← Bouncing  
[●●○○] Understanding Concepts...

Stage 3: ✨ ← Bouncing
[●●●○] Creating Flashcards...

Stage 4: ✅ ← Bouncing
[●●●●] Optimizing Output...
```

### Result Cards (Staggered)
```
Time 0.0s:  Card 1 slides up + fades in
Time 0.1s:  Card 2 slides up + fades in
Time 0.2s:  Card 3 slides up + fades in
Time 0.3s:  Card 4 slides up + fades in
...
```

## Interactive States

### Button States
```
Default:    [  Button  ]
Hover:      [ •Button• ] ← Shadow grows
Active:     [  Button  ] ← Pressed effect
Disabled:   [  Button  ] ← Gray + no hover
Loading:    [ ⟳ Button ] ← Spinning icon
```

### Card Selection
```
Unselected:
┌──────────┐
│ Content  │
└──────────┘

Selected:
┏━━━━━━━━━━┓ ← Purple ring
┃ Content  ┃ ← Purple bg tint
┃    ✓     ┃ ← Check mark
┗━━━━━━━━━━┛
```
